Data Sources
The raw neuroimaging and clinical data are sourced from OpenNeuro.

Dataset ID: ds005917 (v1.0.1) (https://openneuro.org/datasets/ds005917/versions/1.0.1)

Study: "Ketamine-induced changes in functional connectome" (Pharmaco-fMRI cross-over study).

Content Description
1. Raw Data (/raw)
Contains the foundational files for the project.

fMRI Scans: NIfTI files used for signal extraction.

Clinical Metadata: Phenotypic information and depression scale scores used to correlate neural changes with clinical outcomes.

Note: Large neuroimaging files are excluded from the repository via .gitignore.

2. Processed Data (/processed)
This folder contains the results of the notebooks' execution.
